覚え書き
主な処理内容
  • 敵の生成制御などを行う
  • ステージクリア判定
説明
  • ステップとは、1階層内での敵の生成などによる進行の分割 ステップをクリアすると次のステップに行き、 　全てのステップをクリアするとステージクリア
