詳解

オブジェクトの位置や姿勢など、基本的な情報を保持するクラス

行列を管理し、親子関係を保持する。

覚え書き
親子関係などの情報の整合性をとるため、コピーを禁止する。
スケールは行列と別に管理する
todo
行列の計算回数が多いので、速度面の改善が必要

void NObject::CTransform::SetParent ( CTransform * _pParent = nullptr)

親を設定する

警告
階層を深くすると、行列計算が増えるので、注意が必要
  • 特に、ワールド変換行列や座標を取得する関数は呼び出し時に行列を計算するため、 座標の取得のコストが増える
引数
[in]_pParent親へのポインタ(nullptrを設定すると親を解除する) 子オブジェクトの動作
  • 親子関係を設定した時 : 　親の位置にあわされる(ローカル変換行列が単位行列になる) ※ できれば見た目が変わらないようにしたい。
  • 親子関係を解除した時の動作 : 　ワールド空間での座標と姿勢を保持したまま親子関係を解除する(見た目が変わらない)
