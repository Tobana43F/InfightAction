template<typename TAsset, class TInherit>
class NAsset::CFactory< TAsset, TInherit >

アセットの生成と解放を管理する基底クラス

アセットを管理する1つのディレクトリを指定し、 そのディレクトリからの相対パスをKeyとする。 ※ 相対パスは [BindDirectory/]AssetDirectory/Asset.xxx の形式で記述

覚え書き
使用するときは必ずアセットへのスマートポインタで参照カウントの整合性をとること
