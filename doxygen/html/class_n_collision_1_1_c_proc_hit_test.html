詳解

攻撃と被攻撃の当たり判定を行う処理クラス

攻撃側からの攻撃情報を取得し、被攻撃側に受けた攻撃を登録する処理を行う

覚え書き
用途
  • プレイヤーの攻撃と敵の被攻撃判定
  • 敵の攻撃とプレイヤーの被攻撃判定
